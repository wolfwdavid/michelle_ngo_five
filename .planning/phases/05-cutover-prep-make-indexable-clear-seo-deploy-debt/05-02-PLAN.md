<objective>
Clear the three remaining SEO/deploy debt items so the site is cutover-ready. None of these change v1.0 UI behavior; they are wiring/cosmetic edits.

- SEO-05: every page already emits a <title> and an absolute production-host canonical, but NONE emit og:title or og:url. Add both per page (og:title mirrors the page title text, og:url mirrors the page's canonical href) to complete the per-page Open Graph set begun by SEO-01.
- DPLY-01: deploy.yml (staging) runs `node scripts/assert-prerender-count.mjs` after build, but deploy-production.yml (apex) does NOT — a route-count drift could ship to the apex unguarded. Add the identical guard step to the production workflow.

Purpose: Remove the SEO/deploy debt that blocks a clean cutover.
Output: og:title + og:url on all 8 page heads, a prerender-count guard in the production workflow, and a #hero-sentinel in ReelHero.
</objective>
